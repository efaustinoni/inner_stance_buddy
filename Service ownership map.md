# Service ownership map

- quarterService: quarter CRUD and quarter lookup logic
- weekService: week CRUD and week retrieval
- questionService: question retrieval, ordering, import
- answerService: save, update, fetch answers
- trackerService: trackers and check-ins
- dashboardService: dashboard aggregation only
- dashboardOrchestrator: combines multiple services for dashboard flow
- exerciseOrchestrator: combines multi-step exercise flows
